== Description ==

> This plugin provides the sorting and visibility options of product attribute for WooCommerce and Wordpress and also is very lightweight Plugin.
> 

Top Features:

* Dynamic Attribute Sorting.
* Product Attribute Visibility.
* Support With Latest Version.
* Managed With Admin Panel.

Plugin Develop by Vishal Solanki [Visit website](http://vishalksolanki.com/)

== Installation ==

1. Upload the `woo-attribute-global-order-visibility` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin using the 'Plugins' menu in your WordPress admin panel.
3. Go to Product Page then you can see Product Attribute Order Page.
4. Click on that page and set attibute order number and visibility option.
